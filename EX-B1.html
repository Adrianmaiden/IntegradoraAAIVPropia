Ejemplo de Automatización con JavaScript

Este ejemplo muestra cómo cambiar automáticamente el contenido de un elemento en la página:


// Seleccionar un elemento por su ID
const element = document.getElementById('demo');

// Cambiar el contenido del elemento
element.innerText = 'Contenido actualizado automáticamente con JavaScript.';
